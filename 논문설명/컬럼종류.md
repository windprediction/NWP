| `lcc`        | 저층 구름량 (Low Cloud Cover)                           |

| `mcc`        | 대류성 구름량                                            |
| `mgws_0m`    | 지면 최대 돌풍 속도                                        |
| `p`          | 지표면 기압 (Pressure)                                  |
| `pblh`       | 경계층 높이 (Planetary Boundary Layer H                 |
